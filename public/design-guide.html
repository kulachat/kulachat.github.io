Navigation & Overlays

โครงนำทางหลักควรชัดเจนบน desktop และยุบเป็นเมนูที่แตะง่ายบน mobile

Navbar / Sticky Header

แถบเมนูบนสุด ใช้สำหรับ brand, search, navigation และ action หลัก

Hamburger Menu

ปุ่มเมนูสำหรับเปิด mobile drawer หรือ overlay menu

Back to Top Button

floating action สำหรับเอกสารหรือหน้าที่มีเนื้อหายาว

Content containers

ใช้จัดกลุ่มเนื้อหาให้สแกนง่าย เหมาะกับบทความ กิจกรรม บริการ และสถิติ

Service Card

สำหรับบริการหรือ feature หลัก เช่น แบบประเมิน คลังอาชีพ หรือ workshop

Article Card

รองรับ thumbnail, metadata, title และ description

Stat Counter Block

ใช้ตัวเลขเด่นเพื่อสื่อ impact หรือ progress

Media & Assets

visual language ควรเป็นมิตร สว่าง และมี subject ชัดเพื่อช่วยการเรียนรู้

  • Character Illustration mascot หรือภาพประกอบที่ทำให้แบรนด์เข้าถึงง่าย
  • Icon Set line icons ที่ใช้ร่วมกับ label และ navigation ได้ดี
  • Badges / Labels ป้ายบอกหมวดหมู่ สถานะ หรือระดับเนื้อหา

Typography & Grids

ใช้ heading ที่ชัด, body text ที่อ่านง่าย และ grid ที่รองรับทั้ง desktop/mobile

RoleTokenUsage
Heading--ds-font-displayหัวข้อหลักและ section title
Body--ds-font-bodyเนื้อหาและคำอธิบายภาษาไทย
Spacing--ds-space-*ระยะ section, card และ form controls
